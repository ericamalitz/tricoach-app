    🍎 Apple Health / Apple Watch
    Auto-log workouts from your Apple Watch using an iOS Shortcut. Every time you finish a workout it gets sent straight to TriCoach — no manual entry.
    Your personal API token
    Loading...
    Setup (takes 2 min):
    1. Copy your token above
    2. On your iPhone, open the Shortcuts app
    3. Tap + → Add Action → search "Get Last Workout" → add it
    4. Add another action: "Get Details of Workout" — add: Duration, Distance, Active Energy, Average Heart Rate, Workout Type, Start Date
    5. Add action: "Get Contents of URL"
    URL: https://tricoach-server.up.railway.app/api/workout/apple-health?token=YOUR_TOKEN
    Method: POST · Request Body: JSON
    Fields: workoutType, durationSeconds, distanceMeters, avgHeartRate, calories, startDate
    6. Set the Shortcut to run automatically after a workout (Automation → Workout → Ends)
